# Documentation

This folder contains supporting materials for the **Medicare Fraud Data Analysis (Pathology Subset, 2023)** project.

## Dashboard Screenshots
Screenshots highlight key components of the Power BI dashboard, including:
- Summary KPIs
- Top providers by total and average Medicare payments
- Geographic distribution of Medicare payments by state
- Providers flagged by statistical outlier detection methods

These visuals are provided for quick reference without requiring Power BI.

## Methodology Details
The file `methodology.md` provides additional detail on the analytical approach used in this project, including:
- Data filtering and aggregation performed in Python
- Exploratory analysis and visualization steps
- Outlier detection using IQR and Z-score methods
- Interpretation of combined outlier flags

This document complements the high-level methodology described in the main project README.
